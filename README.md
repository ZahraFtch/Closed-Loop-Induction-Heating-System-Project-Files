# Closed-Loop-Induction-Heating-System-Project-Files
Project files for Induction Heating project
This repository contains the project files for Closed-Loop Induction Heating System. Including:
1) STM32 CubeIDE project (sensor and control system)
2) Altium projects (control circuit and power circuit)
3) An Excel file containing the output data of the system in different stages
این پوشه شامل فایلهای پروژه ی کنترل حلقه بسته ی سیستم کوره ی القایی است. شامل:
1) کد میکرو کنترلر STM32
2) فایل پروژه ی Altium (شامل PCB و شماتیک مدار کنترل و قدرت)
3) فایل اکسل دیتاهای ثبت شده از مراحل مختلف
